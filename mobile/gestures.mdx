## Gestes sur la Scène

### Gestes à Un Doigt

<Tabs>
  <Tab title="Tap (Toucher)">
    **Action** : Sélectionner un élément
    
    - Tap sur un danseur : Sélectionne le danseur
    - Tap sur un marqueur : Sélectionne le marqueur
    - Tap sur zone vide : Désélectionne tout
  </Tab>
  
  <Tab title="Glisser">
    **Action** : Déplacer un élément
    
    - Glissez un danseur pour le déplacer
    - Glissez un marqueur pour le repositionner
    - Mouvement fluide et précis
  </Tab>
  
  <Tab title="Appui Long">
    **Action** : Ouvrir le menu contextuel
    
    - Maintenez 1 seconde
    - Menu avec actions disponibles
    - Annulez en glissant hors du menu
  </Tab>
  
  <Tab title="Double Tap">
    **Action** : Zoom rapide
    
    - Double tap sur la scène : Zoom in
    - Double tap à nouveau : Zoom out
    - Centré sur le point tapé
  </Tab>
</Tabs>

### Gestes à Deux Doigts

<Tabs>
  <Tab title="Glisser (Pan)">
    **Action** : Déplacer la vue
    
    - Utilisez 2 doigts pour glisser
    - Déplace la vue de la scène
    - Équivalent du clic droit sur desktop
  </Tab>
  
  <Tab title="Pincement (Zoom)">
    **Action** : Zoomer/Dézoomer
    
    - **Écarter** : Zoom in
    - **Rapprocher** : Zoom out
    - Centré entre les deux doigts
    - Fluide et progressif
  </Tab>
  
  <Tab title="Rotation">
    **Action** : Orienter un danseur
    
    - Sélectionnez un danseur
    - Rotation à 2 doigts pour orienter
    - Angle libre ou par pas de 45°
  </Tab>
</Tabs>

## Gestes sur la Timeline

### Navigation Temporelle

| Geste | Action |
|-------|--------|
| **Tap** | Positionner le playhead |
| **Glisser playhead** | Scrubbing (navigation précise) |
| **Tap sur groupe** | Aller au groupe |
| **Glisser groupe** | Déplacer dans le temps |

### Zoom Temporel

- **Pincement horizontal** : Zoom temporel
- **Double tap** : Ajuster à la vue
- **Glisser 2 doigts** : Défiler horizontalement

## Sélection Multiple

### Sur Scène

<Steps>
  <Step title="Premier élément">
    Tap sur le premier danseur/marqueur
  </Step>
  
  <Step title="Appui long">
    Maintenez le doigt sur l'écran
  </Step>
  
  <Step title="Tap additionnels">
    Avec un autre doigt, tapez sur les autres éléments
  </Step>
  
  <Step title="Relâcher">
    Relâchez pour valider la sélection
  </Step>
</Steps>

### Rectangle de Sélection

<Steps>
  <Step title="Appui long">
    Maintenez dans une zone vide
  </Step>
  
  <Step title="Glisser">
    Glissez pour créer un rectangle
  </Step>
  
  <Step title="Relâcher">
    Tous les éléments dans le rectangle sont sélectionnés
  </Step>
</Steps>

## Gestes Spécifiques

### Tap to Place (Placement Rapide)

Mode spécial pour placer rapidement des danseurs :

<Steps>
  <Step title="Activer le mode">
    Bouton **Tap to Place** dans la barre d'outils
  </Step>
  
  <Step title="Sélectionner un danseur">
    Tap sur un danseur dans la liste
  </Step>
  
  <Step title="Placer sur scène">
    Tap sur la scène à l'emplacement souhaité
  </Step>
  
  <Step title="Répéter">
    Continuez pour placer d'autres danseurs
  </Step>
</Steps>

<Tip>
  Le mode Tap to Place est plus rapide que le glisser-déposer sur mobile.
</Tip>

## Gestes dans les Tiroirs

### Ouvrir/Fermer

- **Swipe depuis le bord** : Ouvre le tiroir
- **Swipe inverse** : Ferme le tiroir
- **Tap en dehors** : Ferme automatiquement

### Navigation

- **Scroll vertical** : Parcourir la liste
- **Tap sur élément** : Sélectionner
- **Glisser élément** : Réorganiser (si applicable)

## Sensibilité et Précision

### Ajuster la Sensibilité

Dans les paramètres :
- **Sensibilité tactile** : Faible, Moyenne, Élevée
- **Seuil de glissement** : Distance minimale pour glisser
- **Durée appui long** : Temps avant menu contextuel

### Améliorer la Précision

<AccordionGroup>
  <Accordion title="Utilisez un Stylet" icon="pen">
    - Plus précis qu'un doigt
    - Idéal pour les placements fins
    - Compatible avec Apple Pencil et stylets Android
  </Accordion>
  
  <Accordion title="Zoomez" icon="magnifying-glass">
    - Zoomez avant de placer
    - Plus facile de viser précisément
    - Utilisez la grille pour l'alignement
  </Accordion>
  
  <Accordion title="Mode Paysage" icon="mobile-screen">
    - Plus d'espace pour travailler
    - Meilleure visibilité
    - Moins de risque de toucher accidentellement
  </Accordion>
</AccordionGroup>

## Gestes à Éviter

### Problèmes Courants

<Warning>
  Certains gestes peuvent causer des actions non désirées :
</Warning>

| Geste | Problème | Solution |
|-------|----------|----------|
| **Tap trop rapide** | Sélection involontaire | Ralentissez légèrement |
| **Glisser trop court** | Interprété comme tap | Glissez plus loin |
| **Pincement asymétrique** | Zoom + rotation | Gardez les doigts parallèles |
| **Appui long trop court** | Pas de menu | Maintenez 1 seconde complète |
