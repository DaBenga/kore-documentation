## Compatibilité

KORE est optimisé pour les appareils mobiles et tablettes.

### Appareils Supportés

<Tabs>
  <Tab title="iPhone">
    - **iOS** : 15+ recommandé
    - **Navigateur** : Safari
    - **Taille** : iPhone 12 et supérieur recommandé
  </Tab>
  
  <Tab title="iPad">
    - **iPadOS** : 15+ recommandé
    - **Navigateur** : Safari
    - **Taille** : Toutes tailles supportées
    - **Expérience optimale** : iPad Pro
  </Tab>
  
  <Tab title="Android">
    - **Version** : Android 10+ recommandé
    - **Navigateur** : Chrome
    - **Taille** : Tablettes 10" recommandées
  </Tab>
</Tabs>

## Interface Mobile

### Adaptations Automatiques

L'interface s'adapte automatiquement :
- **Tiroirs coulissants** : Panneaux latéraux accessibles via boutons
- **Boutons d'action flottants (FAB)** : Accès rapide aux outils
- **Timeline en liste** : Vue simplifiée pour mobile
- **Contrôles tactiles** : Optimisés pour les doigts

### Modes d'Affichage

## Gestes Tactiles

### Gestes de Base

| Geste | Action |
|-------|--------|
| **Tap (1 doigt)** | Sélectionner |
| **Glisser (1 doigt)** | Déplacer danseur/marqueur |
| **Glisser (2 doigts)** | Panoramique scène |
| **Pincement** | Zoom in/out |
| **Appui long** | Menu contextuel |
| **Double tap** | Zoom rapide |

## Boutons d'Action Flottants (FAB)

### Accès Rapide

Les FAB donnent accès aux outils fréquents :
- **Zoom** : Ajuster la vue
- **Grille** : Toggle grille et magnétisme
- **Historique** : Annuler/Refaire
- **Plus** : Menu d'actions supplémentaires

### Position

- **Coin inférieur droit** : Par défaut
- **Déplaçable** : Maintenez et glissez
- **Masquable** : Tap sur la scène pour cacher temporairement

## Tiroirs Latéraux

### Accès aux Tiroirs

- **Bouton hamburger (☰)** : Ouvre le tiroir gauche (Cast/Scènes)
- **Bouton info (ⓘ)** : Ouvre le tiroir droit (Détails scène)
- **Swipe** : Glissez depuis le bord

### Fermeture

- **Tap en dehors** : Ferme le tiroir
- **Swipe inverse** : Glisse pour fermer
- **Bouton X** : Ferme explicitement

## Timeline Mobile

### Vue Liste

Sur mobile, la timeline propose une vue liste :
- **Groupes verticaux** : Plus facile à parcourir
- **Informations détaillées** : Nom, durée, nombre de danseurs
- **Actions rapides** : Éditer, dupliquer, supprimer
- **Navigation** : Tap pour aller au groupe

### Basculer entre Vues

- **Bouton toggle** : Passer de timeline à liste
- **Préférence sauvegardée** : Se souvient de votre choix

## Limitations Mobiles

### Fonctionnalités Réduites

Sur mobile, certaines fonctionnalités sont limitées :
- **Sélection multiple** : Plus difficile (utilisez appui long)
- **Raccourcis clavier** : Non disponibles
- **Multitâche** : Limité par la taille d'écran

### Solutions

- **Utilisez une tablette** : Pour une meilleure expérience
- **Mode paysage** : Plus d'espace
- **Accessoires** : Stylet pour plus de précision
