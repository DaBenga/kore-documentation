## Bienvenue dans KORE !

Ce guide vous accompagne pas à pas pour créer votre premier projet et commencer à chorégraphier.

<Steps>
  <Step title="Créer un compte">
    ### Inscription
    
    1. Rendez-vous sur [kore.app](https://kore.app)
    2. Cliquez sur **"S'inscrire"**
    3. Renseignez votre email et créez un mot de passe
    4. Validez votre compte via l'email de confirmation
    
    <Tip>
      Utilisez une adresse email professionnelle pour faciliter la collaboration avec votre équipe.
    </Tip>
  </Step>

  <Step title="Créer votre premier projet">
    ### Configuration du projet
    
    Une fois connecté, vous arrivez sur le tableau de bord :
    
    1. Cliquez sur le bouton **"+ Nouveau Projet"**
    2. Donnez un nom à votre spectacle (ex: "Gala 2025")
    3. Configurez les dimensions de votre scène :
       - **Largeur** : largeur de la scène (en mètres)
       - **Profondeur** : profondeur de la scène (en mètres)
       - **Largeur des coulisses** : espace côté cour et jardin
    
    <Info>
      Vous pouvez utiliser les préréglages pour les dimensions de scène standard (Opéra Garnier, Châtelet, etc.)
    </Info>
    
    4. Cliquez sur **"Créer"**
  </Step>

  <Step title="Ajouter vos danseurs">
    ### Importer votre casting
    
    Vous avez deux options pour ajouter vos danseurs :
    
    #### Option 1 : Ajout manuel
    
    1. Dans la barre latérale, cliquez sur l'onglet **"Cast"**
    2. Cliquez sur **"+ Ajouter un danseur"**
    3. Renseignez le prénom et le nom
    4. (Optionnel) Ajoutez une photo
    
    #### Option 2 : Import CSV
    
    Pour gagner du temps avec un grand casting :
    
    1. Préparez un fichier CSV avec les colonnes : `prenom`, `nom`, `photo` (URL)
    2. Cliquez sur **"Importer CSV"**
    3. Sélectionnez votre fichier
    4. Validez l'import
    
    <Card title="Format CSV" icon="file-csv">
      ```csv
      prenom,nom,photo
      Marie,Dupont,https://example.com/photo1.jpg
      Jean,Martin,https://example.com/photo2.jpg
      ```
    </Card>
  </Step>

  <Step title="Créer votre première scène">
    ### Organiser votre spectacle
    
    1. Cliquez sur l'onglet **"Scènes"** dans la barre latérale
    2. Cliquez sur **"+ Nouvelle Scène"**
    3. Donnez un nom à la scène (ex: "Ouverture", "Acte 1")
    4. Définissez la durée estimée (en secondes)
    5. (Optionnel) Ajoutez un fichier audio pour synchroniser
    
    <Tip>
      Vous pouvez réorganiser vos scènes par glisser-déposer pour suivre l'ordre du spectacle.
    </Tip>
  </Step>

  <Step title="Placer vos danseurs sur scène">
    ### Créer votre première formation
    
    1. Sélectionnez une scène dans la liste
    2. Dans l'onglet **"Cast"**, assignez les danseurs à cette scène
    3. Passez à l'onglet **"Stage"** pour voir la scène virtuelle
    4. Glissez-déposez les danseurs depuis la barre latérale vers la scène
    5. Positionnez-les précisément en les déplaçant
    
    <Info>
      **Astuce** : Utilisez la grille et le magnétisme (snap to grid) pour aligner vos danseurs facilement.
    </Info>
    
    #### Contrôles de la scène
    
    - **Zoom** : Molette de la souris ou pincement sur mobile
    - **Panoramique** : Clic droit + glisser ou deux doigts sur mobile
    - **Rotation du danseur** : Sélectionnez un danseur et utilisez les boutons de rotation
  </Step>

  <Step title="Organiser dans le temps">
    ### Utiliser la Timeline
    
    1. Cliquez sur l'onglet **"Timeline"** en bas de l'écran
    2. Créez un **groupe de placement** :
       - Cliquez sur **"+ Nouveau Groupe"**
       - Donnez un nom (ex: "Formation 1", "Entrée")
       - Définissez le moment de début et la durée
    3. Les positions actuelles des danseurs sont enregistrées dans ce groupe
    4. Créez d'autres groupes pour les différentes formations
    
    <Tip>
      Vous pouvez naviguer entre les groupes de placement pour voir l'évolution de votre chorégraphie.
    </Tip>
  </Step>
</Steps>
