## Architecture de l'Interface

KORE est organisé en plusieurs zones principales pour optimiser votre flux de travail.

<Frame>
  <img src="/images/interface-overview.png" alt="Vue d'ensemble de l'interface KORE" />
</Frame>

<Note>
  L'interface s'adapte automatiquement à votre appareil (ordinateur, tablette, smartphone).
</Note>

## Zones Principales

### 1. Barre Latérale Gauche (Sidebar)

La barre latérale contient les onglets principaux de navigation :

<Tabs>
  <Tab title="Cast">
    **Gestion du Casting**
    
    - Liste de tous les danseurs du projet
    - Bouton pour ajouter de nouveaux danseurs
    - Import CSV pour ajout en masse
    - Filtrage par scène
    - Indicateurs de changements rapides (Quick Change)
    
    <Tip>
      Les danseurs peuvent être glissés-déposés directement sur la scène depuis cet onglet.
    </Tip>
  </Tab>
  
  <Tab title="Scènes">
    **Liste des Scènes**
    
    - Toutes les scènes de votre spectacle
    - Ordre séquentiel (réorganisable par glisser-déposer)
    - Durée de chaque scène
    - Indicateur de musique associée
    - Bouton pour créer une nouvelle scène
  </Tab>
  
  <Tab title="Stage">
    **Vue Scène**
    
    - Visualisation 2D de la scène
    - Placement des danseurs
    - Contrôles de zoom et panoramique
    - Grille et magnétisme
    - Dimensions et repères
  </Tab>
  
  <Tab title="Drapery">
    **Configuration des Rideaux**
    
    - Position du rideau principal
    - Espacement des pendrillons (legs)
    - Noms des rues
    - Visualisation en temps réel
  </Tab>
  
  <Tab title="Markers">
    **Marqueurs de Scène**
    
    - Groupes de marqueurs
    - Couleurs et formes personnalisables
    - Gestion de la visibilité
    - Verrouillage des groupes
  </Tab>
</Tabs>

### 2. Zone Centrale (Stage)

La zone centrale affiche la **scène virtuelle** où vous positionnez vos danseurs.

#### Éléments de la Scène

<AccordionGroup>
  <Accordion title="Grille" icon="grid">
    - Grille paramétrable (pas de 0.5m à 2m)
    - Subdivisions pour plus de précision
    - Affichage des dimensions
  </Accordion>
  
  <Accordion title="Danseurs" icon="user">
    - Représentation visuelle avec photo ou initiales
    - Flèche d'orientation
    - Nom affiché en permanence
    - Sélection multiple possible
  </Accordion>
  
  <Accordion title="Marqueurs" icon="location-dot">
    - Repères visuels personnalisables
    - Différentes formes (croix, cercle, carré)
    - Couleurs par groupe
  </Accordion>
  
  <Accordion title="Rideaux" icon="curtains">
    - Rideau principal (downstage)
    - Pendrillons (legs) côté cour et jardin
    - Rues numérotées
  </Accordion>
</AccordionGroup>

#### Contrôles de la Scène

| Action | Ordinateur | Mobile/Tablette |
|--------|-----------|-----------------|
| **Zoom** | Molette souris | Pincement (2 doigts) |
| **Panoramique** | Clic droit + glisser | Glisser (2 doigts) |
| **Déplacer danseur** | Glisser-déposer | Glisser (1 doigt) |
| **Sélection multiple** | Ctrl/Cmd + clic | Appui long |
| **Rotation danseur** | Boutons rotation | Boutons rotation |

### 3. Barre d'Outils (Toolbar)

La barre d'outils en haut de la scène contient les actions rapides :

<CardGroup cols={2}>
  <Card title="Zoom" icon="magnifying-glass">
    - Zoom avant/arrière
    - Ajuster à la scène
    - Réinitialiser le zoom
  </Card>
  
  <Card title="Grille" icon="grid-2">
    - Activer/désactiver la grille
    - Activer/désactiver le magnétisme
    - Ajuster le pas de la grille
  </Card>
  
  <Card title="Affichage" icon="eye">
    - Afficher/masquer les danseurs
    - Afficher/masquer les photos
    - Ajuster la taille des danseurs
  </Card>
  
  <Card title="Historique" icon="clock-rotate-left">
    - Annuler (Ctrl+Z)
    - Refaire (Ctrl+Y)
    - Historique des modifications
  </Card>
</CardGroup>

### 4. Panneau Latéral Droit (SidePanel)

Le panneau droit affiche des informations contextuelles selon la scène sélectionnée :

- **Casting de la scène** : danseurs assignés à cette scène
- **Détection Quick Change** : alertes pour changements rapides
- **Propriétés de la scène** : durée, audio, paramètres

<Info>
  Sur mobile, ce panneau devient un tiroir coulissant accessible via un bouton.
</Info>

### 5. Timeline (Bas de l'Écran)

La timeline permet d'organiser vos chorégraphies dans le temps.

#### Composants de la Timeline

<Steps>
  <Step title="Waveform Audio">
    Visualisation de la forme d'onde si un fichier audio est associé à la scène
  </Step>
  
  <Step title="Groupes de Placement">
    Blocs représentant les différentes formations/positions dans le temps
  </Step>
  
  <Step title="Playhead">
    Curseur de lecture pour naviguer dans le temps
  </Step>
  
  <Step title="Contrôles">
    - Lecture/Pause
    - Zoom temporel
    - Navigation entre groupes
  </Step>
</Steps>

#### Vue Liste (Mobile)

Sur mobile, une **vue liste** alternative permet de gérer les groupes de placement plus facilement :

- Liste verticale des groupes
- Informations détaillées par groupe
- Actions rapides (éditer, dupliquer, supprimer)

## Modes d'Affichage

### Mode Bureau (Desktop)

- Toutes les zones visibles simultanément
- Barre latérale fixe à gauche
- Panneau droit fixe
- Timeline en bas

### Mode Tablette

- Interface adaptée au tactile
- Tiroirs coulissants pour les panneaux latéraux
- Boutons d'action flottants (FAB)
- Gestes tactiles optimisés

### Mode Mobile

- Interface simplifiée
- Navigation par onglets
- Tiroirs plein écran
- Vue liste pour la timeline
- Contrôles tactiles dédiés

<Tip>
  Utilisez le bouton **Plein Écran** pour maximiser la zone de scène et gagner en espace de travail.
</Tip>

## Navigation Rapide

### Raccourcis Clavier

| Raccourci | Action |
|-----------|--------|
| `Ctrl/Cmd + Z` | Annuler |
| `Ctrl/Cmd + Y` | Refaire |
| `Suppr` | Supprimer la sélection |
| `Ctrl/Cmd + A` | Tout sélectionner |
| `Espace` | Lecture/Pause |
| `←` `→` | Groupe précédent/suivant |
