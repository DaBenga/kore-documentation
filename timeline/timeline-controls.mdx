## Interface de la Timeline

La timeline se trouve en bas de l'écran et affiche :
- **Waveform** : Forme d'onde audio (si présente)
- **Groupes de placement** : Blocs colorés
- **Playhead** : Curseur de lecture (ligne rouge)
- **Marqueurs** : Repères temporels
- **Contrôles** : Boutons de lecture et zoom

## Playhead (Tête de Lecture)

### Positionner le Playhead

| Méthode | Action |
|---------|--------|
| **Clic** | Cliquez sur la timeline |
| **Glisser** | Faites glisser le playhead |
| **Raccourcis** | `←` `→` pour avancer/reculer |

### Synchronisation

Le playhead est synchronisé avec :
- La position audio (si présente)
- Les positions des danseurs sur scène
- Les groupes de placement actifs

## Lecture et Contrôles

### Boutons de Lecture

<Tabs>
  <Tab title="Play/Pause">
    - **Bouton** : ▶️ / ⏸️
    - **Raccourci** : `Espace`
    - Lance ou met en pause la lecture
  </Tab>
  
  <Tab title="Stop">
    - **Bouton** : ⏹️
    - **Raccourci** : `S`
    - Arrête et revient au début
  </Tab>
  
  <Tab title="Précédent/Suivant">
    - **Boutons** : ⏮️ / ⏭️
    - **Raccourcis** : `←` / `→`
    - Navigue entre les groupes
  </Tab>
</Tabs>

### Vitesse de Lecture

Ajustez la vitesse pour l'analyse :
- **0.25x** : Très lent
- **0.5x** : Ralenti
- **1.0x** : Normal
- **1.5x** : Rapide
- **2.0x** : Très rapide

<Tip>
  La vitesse ralentie est utile pour analyser les transitions complexes.
</Tip>

## Zoom Temporel

### Zoomer sur la Timeline

| Action | Méthode |
|--------|---------|
| **Zoom In** | Molette haut ou `+` |
| **Zoom Out** | Molette bas ou `-` |
| **Ajuster** | Bouton "Fit" ou `F` |
| **Zoom sur sélection** | Double-clic sur une zone |

### Niveaux de Zoom

- **Vue d'ensemble** : Toute la scène visible
- **Vue détaillée** : Secondes visibles
- **Vue précise** : Dixièmes de seconde

## Navigation

### Défilement Horizontal

- **Glisser** : Cliquez et glissez sur la timeline
- **Barre de défilement** : En bas de la timeline
- **Raccourcis** : `Shift + Molette` pour défiler

### Aller à un Moment Précis

1. Cliquez sur le temps affiché
2. Entrez le moment souhaité (format: `MM:SS`)
3. Appuyez sur `Entrée`

## Marqueurs Temporels

### Ajouter un Marqueur

<Steps>
  <Step title="Positionner le playhead">
    Au moment souhaité
  </Step>
  
  <Step title="Créer le marqueur">
    Cliquez sur **+ Marqueur** ou `M`
  </Step>
  
  <Step title="Nommer">
    Donnez un nom (ex: "Climax", "Changement")
  </Step>
  
  <Step title="Personnaliser">
    Choisissez une couleur
  </Step>
</Steps>

### Naviguer avec les Marqueurs

- **Clic** : Aller directement au marqueur
- **Raccourcis** : `Shift + ←/→` pour marqueur précédent/suivant

## Sélection de Zone

### Sélectionner une Plage

1. Cliquez sur le début de la plage
2. `Shift + Clic` sur la fin
3. La zone est surlignée

### Actions sur Sélection

- **Zoom** : Double-clic pour zoomer sur la sélection
- **Lecture** : Lire uniquement la sélection
- **Export** : Exporter cette portion

## Affichage

### Options d'Affichage

Personnalisez la timeline :

| Option | Description |
|--------|-------------|
| **Waveform** | Afficher/masquer la forme d'onde |
| **Noms des groupes** | Afficher/masquer les labels |
| **Marqueurs** | Afficher/masquer les marqueurs |
| **Grille temporelle** | Lignes de repère |

### Hauteur de la Timeline

Ajustez la hauteur :
- **Glisser** : Bord supérieur de la timeline
- **Préréglages** : Petite, Moyenne, Grande
- **Plein écran** : Maximiser la timeline

## Raccourcis Clavier

| Raccourci | Action |
|-----------|--------|
| `Espace` | Play/Pause |
| `S` | Stop |
| `←` / `→` | Groupe précédent/suivant |
| `Shift + ←/→` | Marqueur précédent/suivant |
| `+` / `-` | Zoom in/out |
| `F` | Ajuster à la vue |
| `M` | Ajouter un marqueur |
| `Home` | Aller au début |
| `End` | Aller à la fin |

## Synchronisation Audio

### Lecture avec Audio

Si un fichier audio est associé :
- La lecture est synchronisée
- Le playhead suit la musique
- Les groupes s'affichent au bon moment

### Contrôles Audio

- **Volume** : Curseur de volume
- **Mute** : `Ctrl/Cmd + M`
- **Vitesse** : Affecte aussi l'audio
