## Accéder aux Paramètres

Pour modifier les paramètres d'un projet existant :

1. Ouvrez le projet
2. Cliquez sur l'icône **Paramètres** (⚙️) dans la barre d'outils
3. Ou utilisez le menu projet > **Paramètres**

## Informations Générales

### Nom du Projet

- Modifiez le nom de votre spectacle à tout moment
- Le nom apparaît dans la liste des projets et l'en-tête

### Métadonnées

- **Date de création** : Affichée automatiquement
- **Dernière modification** : Mise à jour automatiquement
- **Créateur** : Utilisateur ayant créé le projet

## Dimensions de Scène

### Modifier les Dimensions

Vous pouvez ajuster les dimensions même après la création :

<Warning>
  Modifier les dimensions peut affecter le positionnement des danseurs existants. Vérifiez vos placements après modification.
</Warning>

| Paramètre | Modification |
|-----------|--------------|
| **Largeur de scène** | Ajustez en mètres |
| **Profondeur de scène** | Ajustez en mètres |
| **Largeur des coulisses** | Espace cour/jardin |

### Préréglages

<Steps>
  <Step title="Créer un préréglage">
    1. Configurez vos dimensions
    2. Cliquez sur **"Sauvegarder comme préréglage"**
    3. Donnez un nom au préréglage
  </Step>
  
  <Step title="Utiliser un préréglage">
    1. Cliquez sur **"Charger un préréglage"**
    2. Sélectionnez dans la liste
    3. Les dimensions sont appliquées automatiquement
  </Step>
  
  <Step title="Supprimer un préréglage">
    1. Accédez à la liste des préréglages
    2. Cliquez sur l'icône de suppression
    3. Confirmez la suppression
  </Step>
</Steps>

## Configuration de la Grille

### Paramètres de Grille

- **Pas de la grille** : De 0.5m à 2m
- **Subdivisions** : Lignes intermédiaires pour plus de précision
- **Couleur** : Personnalisez l'apparence

### Magnétisme (Snap to Grid)

- **Activer/Désactiver** : Par défaut désactivé
- **Force du magnétisme** : Ajustez la sensibilité
- **Seuil** : Distance d'accrochage en pixels

<Tip>
  Le magnétisme facilite l'alignement des danseurs sur la grille.
</Tip>

## Paramètres d'Affichage

### Taille des Danseurs

Ajustez la taille d'affichage des danseurs sur scène :

- **Petite** : Pour les grandes formations
- **Moyenne** : Taille par défaut
- **Grande** : Pour plus de détails

### Visibilité

Contrôlez ce qui est affiché sur la scène :

<Tabs>
  <Tab title="Danseurs">
    - Afficher/masquer tous les danseurs
    - Afficher/masquer les photos
    - Afficher/masquer les noms
    - Afficher/masquer les flèches d'orientation
  </Tab>
  
  <Tab title="Marqueurs">
    - Afficher/masquer par groupe
    - Verrouiller les groupes
    - Ajuster la taille
  </Tab>
  
  <Tab title="Rideaux">
    - Afficher/masquer les pendrillons
    - Afficher/masquer le rideau principal
    - Afficher/masquer les noms de rues
  </Tab>
  
  <Tab title="Grille">
    - Afficher/masquer la grille
    - Afficher/masquer les dimensions
    - Afficher/masquer les repères
  </Tab>
</Tabs>

## Duplication de Projet

### Dupliquer un Projet Existant

Pour créer une copie d'un projet :

<Steps>
  <Step title="Accéder à la duplication">
    Dans la liste des projets, cliquez sur le menu (⋮) > **Dupliquer**
  </Step>
  
  <Step title="Choisir les éléments">
    Sélectionnez ce que vous souhaitez copier :
    - ✅ Danseurs et casting
    - ✅ Scènes et structure
    - ✅ Placements et formations
    - ✅ Marqueurs
    - ✅ Configuration des rideaux
  </Step>
  
  <Step title="Nommer la copie">
    Donnez un nom au nouveau projet (ex: "Gala 2025 - Copie")
  </Step>
  
  <Step title="Créer">
    Cliquez sur **Dupliquer** pour créer la copie
  </Step>
</Steps>

<Info>
  La duplication est utile pour créer des variantes ou travailler sur différentes versions d'un spectacle.
</Info>

## Suppression de Projet

### Supprimer un Projet

<Warning>
  La suppression est **irréversible**. Toutes les données du projet seront perdues définitivement.
</Warning>

Pour supprimer un projet :

1. Dans la liste des projets, cliquez sur le menu (⋮)
2. Sélectionnez **Supprimer**
3. Confirmez en tapant le nom du projet
4. Cliquez sur **Supprimer définitivement**

## Paramètres Avancés

### Historique

- **Nombre d'annulations** : Limite de l'historique (défaut : 50)
- **Sauvegardes automatiques** : Fréquence des sauvegardes

### Performance

- **Qualité de rendu** : Ajustez selon votre appareil
- **Animations** : Activer/désactiver les transitions
- **Préchargement** : Charger les ressources en avance
