## Créer un Nouveau Projet

Un projet dans KORE représente un spectacle complet avec ses scènes, son casting et ses chorégraphies.

### Étapes de Création

<Steps>
  <Step title="Accéder à la création">
    Depuis le tableau de bord, cliquez sur le bouton **"+ Nouveau Projet"**
  </Step>
  
  <Step title="Nommer le projet">
    Donnez un nom significatif à votre spectacle :
    - Nom du spectacle (ex: "Gala 2025", "Le Lac des Cygnes")
    - Saison ou date si pertinent
  </Step>
  
  <Step title="Configurer la scène">
    Définissez les dimensions de votre espace scénique
  </Step>
</Steps>

## Configuration de la Scène

### Dimensions Personnalisées

Renseignez les mesures de votre scène en mètres :

| Paramètre | Description | Valeur typique |
|-----------|-------------|----------------|
| **Largeur** | Largeur totale de la scène (ouverture) | 12-20m |
| **Profondeur** | Distance du bord de scène au fond | 10-15m |
| **Largeur des coulisses** | Espace côté cour et jardin | 3-5m |

<Tip>
  Les dimensions influencent la visualisation et les calculs de placement. Soyez aussi précis que possible.
</Tip>

### Préréglages de Scène

KORE propose des préréglages pour les scènes célèbres :

<AccordionGroup>
  <Accordion title="Opéra Garnier" icon="building-columns">

    - Profondeur : 14m
    - Coulisses : 4m

  <Accordion title="Théâtre du Châtelet" icon="building-columns">

  <Accordion title="Scène Standard" icon="building-columns">

<Info>
  Vous pouvez créer et sauvegarder vos propres préréglages pour réutilisation future.
</Info>

## Configuration des Rideaux

Lors de la création, vous pouvez également configurer :

### Rideau Principal

- **Distance du bord de scène** : Position du rideau d'ouverture (downstage)
- Généralement entre 1m et 3m du bord

### Pendrillons (Legs)

- **Nombre de pendrillons** : Nombre de paires de rideaux latéraux
- **Espacement** : Distance entre chaque paire
- **Noms des rues** : Numérotation ou noms personnalisés

<Card title="En savoir plus" icon="curtains" href="/stage/drapery">
  Consultez la documentation complète sur la configuration des rideaux
</Card>

## Validation et Création

Une fois tous les paramètres renseignés :

1. Vérifiez les informations
2. Cliquez sur **"Créer le projet"**
3. Vous êtes redirigé vers votre nouveau projet

<Success>
  Votre projet est créé ! Vous pouvez maintenant ajouter des danseurs et créer des scènes.
</Success>

## Conseils

<AccordionGroup>
  <Accordion title="Nommage des Projets" icon="tag">
    Utilisez une convention de nommage claire :
    - Incluez la saison ou l'année
    - Soyez descriptif mais concis
    - Exemples : "Gala 2025", "Casse-Noisette Déc2024"
  </Accordion>
  
  <Accordion title="Mesures Précises" icon="ruler">
    Pour obtenir les dimensions exactes de votre scène :
    - Consultez les plans techniques du théâtre
    - Mesurez sur place si possible
    - Contactez le régisseur de la salle
  </Accordion>
  
  <Accordion title="Préréglages Personnalisés" icon="bookmark">
    Si vous travaillez régulièrement dans la même salle :
    - Créez un préréglage personnalisé
    - Sauvegardez-le pour réutilisation
    - Gagnez du temps sur les futurs projets
  </Accordion>
</AccordionGroup>
