---

L'onglet **Habillage** (Drapery) vous permet de construire l'espace physique de votre scène. C'est ici que vous définissez "la boîte noire".

## Éléments Configurables

Vous pouvez ajouter et positionner :

- **Rideau d'avant-scène** : Le rideau principal (rouge ou noir).
- **Pendrillons (Legs)** : Les bandes de tissu verticales sur les côtés qui cachent les coulisses.
- **Fonds (Cyclorama / Mur)** : Le fond de scène.

## Personnalisation

Pour chaque pendrillon, vous pouvez régler :

- La position (distance depuis le cadre de scène).
- La largeur et la hauteur.
- L'ouverture (pour déterminer la largeur de la scène visible).

Ces éléments définissent l'espace visible sur votre plan de scène.

## Exports

Vous pouvez exporter votre configuration sous forme de **Plan de Scène** (PDF) incluant toutes les cotes et distances pour l'équipe technique du théâtre.
