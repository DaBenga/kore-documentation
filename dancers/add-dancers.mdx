## Méthodes d'Ajout

KORE propose deux méthodes pour ajouter des danseurs à votre projet.

### Ajout Manuel

Pour ajouter un danseur individuellement :

<Steps>
  <Step title="Accéder à l'onglet Cast">
    Cliquez sur **Cast** dans la barre latérale gauche
  </Step>
  
  <Step title="Créer un nouveau danseur">
    Cliquez sur le bouton **+ Ajouter un danseur**
  </Step>
  
  <Step title="Renseigner les informations">
    - **Prénom** : Prénom du danseur (requis)
    - **Nom** : Nom de famille (requis)
    - **Photo** : Image de profil (optionnel)
  </Step>
  
  <Step title="Valider">
    Cliquez sur **Créer** pour ajouter le danseur
  </Step>
</Steps>

<Tip>
  L'ajout manuel est idéal pour les petits castings ou les ajouts ponctuels.
</Tip>

### Import CSV

Pour importer plusieurs danseurs en une fois :

<Steps>
  <Step title="Préparer le fichier CSV">
    Créez un fichier CSV avec les colonnes suivantes :
    - `prenom` (requis)
    - `nom` (requis)
    - `photo` (optionnel - URL de l'image)
    - `scenes` (optionnel - noms des scènes séparés par des virgules)
  </Step>
  
  <Step title="Lancer l'import">
    1. Cliquez sur **Importer CSV** dans l'onglet Cast
    2. Sélectionnez votre fichier
    3. Vérifiez l'aperçu des données
  </Step>
  
  <Step title="Configurer l'import">
    Choisissez les options :
    - ✅ Importer les danseurs
    - ✅ Importer les scènes (si présentes dans le CSV)
    - ✅ Assigner automatiquement aux scènes
  </Step>
  
  <Step title="Valider l'import">
    Cliquez sur **Importer** pour ajouter tous les danseurs
  </Step>
</Steps>

#### Format du Fichier CSV

```csv
prenom,nom,photo,scenes
Marie,Dupont,https://example.com/marie.jpg,"Acte 1,Finale"
Jean,Martin,https://example.com/jean.jpg,"Acte 1,Acte 2"
Sophie,Bernard,,Finale
```

<Info>
  Les URLs de photos doivent être accessibles publiquement. Vous pouvez utiliser des services comme Imgur, Google Drive (lien public), ou votre propre serveur.
</Info>

## Ajouter des Photos

### Photo lors de la Création

Lors de l'ajout manuel, vous pouvez :

1. **Télécharger une image** depuis votre ordinateur
2. **Coller une URL** d'image en ligne
3. **Laisser vide** : les initiales seront affichées

### Ajouter/Modifier une Photo Ultérieurement

<Steps>
  <Step title="Sélectionner le danseur">
    Cliquez sur le danseur dans la liste du Cast
  </Step>
  
  <Step title="Modifier">
    Cliquez sur l'icône d'édition (crayon)
  </Step>
  
  <Step title="Changer la photo">
    - Téléchargez une nouvelle image
    - Ou collez une nouvelle URL
    - Ou supprimez la photo existante
  </Step>
  
  <Step title="Sauvegarder">
    Cliquez sur **Enregistrer**
  </Step>
</Steps>

### Recommandations pour les Photos

<AccordionGroup>
  <Accordion title="Format et Taille" icon="image">
    - **Format** : JPG, PNG, ou WebP
    - **Taille recommandée** : 200x200 pixels minimum
    - **Poids** : Moins de 1 MB pour de meilleures performances
  </Accordion>
  
  <Accordion title="Qualité" icon="star">
    - Photo de face, bien éclairée
    - Fond neutre de préférence
    - Visage bien visible
    - Format carré idéal
  </Accordion>
  
  <Accordion title="Confidentialité" icon="lock">
    - Assurez-vous d'avoir l'autorisation d'utiliser les photos
    - Respectez le droit à l'image
    - Utilisez des URLs sécurisées (HTTPS)
  </Accordion>
</AccordionGroup>

## Gestion en Masse

### Importer depuis une Feuille de Calcul

Si vous gérez votre casting dans Excel ou Google Sheets :

1. **Exportez en CSV** depuis votre tableur
2. Vérifiez que les colonnes correspondent au format KORE
3. Importez le fichier dans KORE

### Mise à Jour en Masse

Pour mettre à jour plusieurs danseurs :

1. Exportez la liste actuelle (fonctionnalité à venir)
2. Modifiez le CSV
3. Réimportez avec l'option **"Mettre à jour les existants"**

<Warning>
  L'import CSV avec mise à jour écrasera les données existantes. Faites une sauvegarde avant.
</Warning>

## Conseils

<Tabs>
  <Tab title="Organisation">
    **Nommage Cohérent**
    
    - Utilisez toujours Prénom + Nom
    - Évitez les surnoms dans les champs officiels
    - Soyez cohérent avec les majuscules
  </Tab>
  
  <Tab title="Photos">
    **Hébergement des Images**
    
    Options pour héberger vos photos :
    - Google Drive (liens publics)
    - Imgur (gratuit, facile)
    - Dropbox (liens publics)
    - Votre propre serveur web
  </Tab>
  
  <Tab title="Import">
    **Préparation du CSV**
    
    - Utilisez un éditeur de texte ou Excel
    - Encodage UTF-8 pour les accents
    - Vérifiez les virgules dans les noms
    - Testez avec quelques lignes d'abord
  </Tab>
</Tabs>
