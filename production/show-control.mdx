title: "Régie"
description: "Le mode spectacle pour lancer votre conduite."
---

L'espace **Régie** (Show Control) est polyvalent. Il permet à la fois de rédiger la conduite technique et de piloter le spectacle.

## Rédaction de la Conduite

C'est ici que vous formalisez la conduite pour les techniciens.

- **Ajout de Tops** : Détaillez les actions attendues (Gros titres, sous-titres, notes techniques).
- **Mise en page** : Adaptez l'affichage pour qu'il soit lisible dans le noir.

## Exports

Si vous ne souhaitez pas utiliser l'application en direct, vous pouvez exporter votre travail :

- **PDF** : Pour une impression papier propre et formatée de la conduite.
- **GrandMA** : Exportez directement votre séquence lumière (format XML) pour l'importer dans la console GrandMA.

## Mode Spectacle (Live)

Pour le "Topeur" ou le Régisseur Général qui souhaite lancer le spectacle depuis l'application :

- **Filage interactif** : Suivez l'avancement tableau par tableau.
- **Chronomètre** : Lancez le timer au début de chaque tableau.
