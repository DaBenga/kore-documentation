Une fois vos tableaux créés, la vue **Durées et Ordre** vous permet d'affiner le déroulé temporel.

## Gestion du Temps

Pour chaque tableau, vous pouvez préciser :

- **Durée estimée** : Le temps prévu pour cette séquence.
- **Durée réelle** : Le temps chronométré (utile lors des répétitions).

Koré calcule automatiquement la durée totale et vous indique l'écart par rapport à votre durée cible.

## Organisation par Actes

Ici, vous organisez la structure définie précédemment :

- **Assigner** : Glissez vos tableaux dans les Actes correspondants.
- **Ordonner** : Définissez l'ordre des Actes et des Entractes dans le déroulé global.

## Optimisation de l'Ordre (Quick Changes)

Koré vous aide à trouver le meilleur enchaînement pour minimiser les changements rapides (Quick Changes) compliqués.

### Verrouillage de Groupes (Link)

Vous pouvez **lier** plusieurs tableaux ensemble (ex: "Tableau 1" doit absolument être suivi de "Tableau 2"). Ces groupes de tableaux se déplaceront d'un bloc si vous réorganisez le spectacle.

### Propositions d'Ordre

Si des conflits de costumes ou de temps de changement sont détectés, Koré peut vous suggérer un **ordre optimisé** des tableaux pour résoudre ces problèmes automatiquement.

## Exports

Une fois l'ordre validé, vous pouvez imprimer la **Conduite Simple** (PDF) pour l'afficher en coulisses.
