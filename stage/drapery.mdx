## Système de Rideaux

KORE permet de configurer les rideaux de scène pour une visualisation réaliste :
- **Rideau principal** (downstage curtain)
- **Pendrillons** (legs) côté cour et jardin
- **Rues** (streets) entre les pendrillons

## Rideau Principal

### Position

Le rideau principal se situe en avant-scène :
- **Distance du bord** : 0m à 5m du downstage
- **Valeur typique** : 1-2m
- **Ajustable** : Via le panneau Drapery

### Configuration

<Steps>
  <Step title="Ouvrir Drapery">
    Cliquez sur l'onglet **Drapery** dans la barre latérale
  </Step>
  
  <Step title="Ajuster la distance">
    Utilisez le curseur ou entrez une valeur en mètres
  </Step>
  
  <Step title="Visualiser">
    Le rideau se déplace en temps réel sur la scène
  </Step>
</Steps>

## Pendrillons (Legs)

### Qu'est-ce qu'un Pendrillon ?

Les pendrillons sont des rideaux verticaux placés :
- De chaque côté de la scène (cour et jardin)
- À intervalles réguliers
- Pour masquer les coulisses

### Configuration des Pendrillons

<Tabs>
  <Tab title="Nombre">
    Définissez le nombre de paires de pendrillons :
    - **Minimum** : 2 paires
    - **Maximum** : 8 paires
    - **Standard** : 4-5 paires
  </Tab>
  
  <Tab title="Espacement">
    Distance entre chaque paire :
    - **Uniforme** : Même espacement partout
    - **Personnalisé** : Espacement variable
    - **Valeur typique** : 2-4m
  </Tab>
  
  <Tab title="Largeur">
    Largeur de chaque pendrillon :
    - Généralement 1-2m
    - Ajustable individuellement
  </Tab>
</Tabs>

### Espacement Personnalisé

Pour un espacement variable :

1. Désactivez **"Espacement uniforme"**
2. Ajustez chaque intervalle individuellement
3. Visualisez en temps réel

<Info>
  L'espacement personnalisé est utile pour adapter la configuration à des scènes spécifiques ou des besoins artistiques particuliers.
</Info>

## Rues (Streets)

### Qu'est-ce qu'une Rue ?

Les rues sont les espaces entre les pendrillons :
- **Rue 0** : Avant le rideau principal
- **Rue 1** : Entre rideau et 1er pendrillon
- **Rue 2** : Entre 1er et 2ème pendrillon
- Etc.

### Nommer les Rues

Personnalisez les noms pour faciliter la communication :

<Steps>
  <Step title="Activer les noms">
    Cochez **"Afficher les noms de rues"**
  </Step>
  
  <Step title="Modifier un nom">
    Cliquez sur le nom de la rue pour l'éditer
  </Step>
  
  <Step title="Conventions">
    Utilisez une convention claire :
    - Numérotation : "Rue 1", "Rue 2"
    - Descriptif : "Avant-scène", "Milieu", "Fond"
    - Mixte : "R1 - Avant", "R2 - Centre"
  </Step>
</Steps>

## Visualisation

### Affichage sur la Scène

Les rideaux sont représentés par :
- **Lignes verticales** : Pendrillons
- **Ligne horizontale** : Rideau principal
- **Labels** : Noms des rues
- **Couleur** : Personnalisable

### Options d'Affichage

Contrôlez la visibilité :
- **Afficher/Masquer les pendrillons**
- **Afficher/Masquer le rideau principal**
- **Afficher/Masquer les noms de rues**
- **Ajuster l'opacité**

## Préréglages

### Sauvegarder une Configuration

Pour réutiliser une configuration :

1. Configurez vos rideaux
2. Cliquez sur **"Sauvegarder comme préréglage"**
3. Donnez un nom (ex: "Opéra Garnier", "Théâtre Municipal")
4. Le préréglage est sauvegardé

### Charger un Préréglage

1. Cliquez sur **"Charger un préréglage"**
2. Sélectionnez dans la liste
3. La configuration est appliquée automatiquement

## Utilisation avec les Danseurs

### Placement par Rue

Les rues facilitent le placement :
- "Placez-vous en rue 2"
- "Entrée par rue 3 côté jardin"
- "Formation en rue 1"

### Visibilité

Considérez les pendrillons pour :
- Les entrées et sorties
- Les zones masquées au public
- Les changements de costume
- Les effets de surprise

<Tip>
  Utilisez les noms de rues dans vos notes de mise en scène pour une communication claire avec l'équipe.
</Tip>
