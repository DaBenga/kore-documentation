## Prérequis

Avant de placer des danseurs sur scène :
- ✅ Créez une scène
- ✅ Ajoutez des danseurs au projet
- ✅ Assignez les danseurs à la scène

## Placer un Danseur

### Méthode 1 : Glisser-Déposer

<Steps>
  <Step title="Sélectionner la scène">
    Cliquez sur la scène dans l'onglet Scènes
  </Step>
  
  <Step title="Passer en vue Stage">
    Cliquez sur l'onglet **Stage** pour voir la scène virtuelle
  </Step>
  
  <Step title="Glisser le danseur">
    Depuis la barre latérale, glissez un danseur vers la scène
  </Step>
  
  <Step title="Positionner">
    Relâchez à l'emplacement souhaité
  </Step>
</Steps>

### Méthode 2 : Tap to Place (Mobile)

Sur mobile et tablette :
1. Sélectionnez un danseur dans la liste
2. Tapez sur la scène à l'emplacement désiré
3. Le danseur apparaît à cette position

## Déplacer un Danseur

### Sur Ordinateur

- **Cliquez et glissez** : Maintenez le clic et déplacez
- **Touches fléchées** : Déplacement précis (1 pixel)
- **Shift + Flèches** : Déplacement rapide (10 pixels)

### Sur Mobile/Tablette

- **Glisser avec un doigt** : Déplacez le danseur
- **Magnétisme** : Active le snap to grid automatiquement

<Tip>
  Activez le magnétisme (snap to grid) pour aligner facilement les danseurs sur la grille.
</Tip>

## Orienter un Danseur

Chaque danseur a une **flèche d'orientation** indiquant sa direction.

### Modifier l'Orientation

<Tabs>
  <Tab title="Boutons de Rotation">
    1. Sélectionnez le danseur
    2. Utilisez les boutons de rotation dans la barre d'outils
    3. Rotation par pas de 45°
  </Tab>
  
  <Tab title="Rotation Manuelle">
    1. Sélectionnez le danseur
    2. Cliquez sur la flèche d'orientation
    3. Faites glisser pour orienter librement
  </Tab>
</Tabs>

### Angles Prédéfinis

| Angle | Direction | Usage |
|-------|-----------|-------|
| 0° | Face public | Position standard |
| 45° | Diagonale | Formations en V |
| 90° | Côté cour/jardin | Profil |
| 180° | Dos au public | Effets dramatiques |

## Sélection Multiple

### Sélectionner Plusieurs Danseurs

<AccordionGroup>
  <Accordion title="Sélection par Rectangle" icon="square-dashed">
    1. Cliquez dans une zone vide
    2. Glissez pour créer un rectangle de sélection
    3. Tous les danseurs dans le rectangle sont sélectionnés
  </Accordion>
  
  <Accordion title="Sélection Additive" icon="plus">
    - **Ctrl/Cmd + Clic** : Ajouter à la sélection
    - **Shift + Clic** : Sélectionner une plage
  </Accordion>
  
  <Accordion title="Tout Sélectionner" icon="check-double">
    - **Ctrl/Cmd + A** : Sélectionne tous les danseurs sur scène
  </Accordion>
</AccordionGroup>

### Actions sur Sélection Multiple

Avec plusieurs danseurs sélectionnés :
- **Déplacer ensemble** : Glissez pour déplacer le groupe
- **Aligner** : Utilisez les outils d'alignement
- **Orienter** : Rotation simultanée
- **Supprimer** : Retirer de la scène

## Outils d'Alignement

### Aligner les Danseurs

<Tabs>
  <Tab title="Alignement Horizontal">
    - **Haut** : Aligne sur le danseur le plus haut
    - **Centre** : Aligne au centre vertical
    - **Bas** : Aligne sur le danseur le plus bas
  </Tab>
  
  <Tab title="Alignement Vertical">
    - **Gauche** : Aligne sur le danseur le plus à gauche
    - **Centre** : Aligne au centre horizontal
    - **Droite** : Aligne sur le danseur le plus à droite
  </Tab>
  
  <Tab title="Distribution">
    - **Espacement horizontal** : Répartit uniformément en largeur
    - **Espacement vertical** : Répartit uniformément en profondeur
  </Tab>
</Tabs>

## Grille et Magnétisme

### Activer la Grille

La grille aide à positionner précisément :
- **Afficher/Masquer** : Bouton dans la barre d'outils
- **Pas de grille** : De 0.5m à 2m
- **Subdivisions** : Lignes intermédiaires

### Snap to Grid (Magnétisme)

Quand activé :
- Les danseurs s'accrochent automatiquement à la grille
- Facilite l'alignement
- Ajustable en sensibilité

<Info>
  Le magnétisme est désactivé par défaut. Activez-le pour des formations géométriques précises.
</Info>

## Retirer de la Scène

Pour retirer un danseur de la scène :

1. Sélectionnez le danseur
2. Appuyez sur **Suppr** ou cliquez sur l'icône de suppression
3. Le danseur reste dans le casting mais n'est plus positionné

<Warning>
  Retirer un danseur de la scène ne le supprime pas du projet, seulement de cette position spécifique.
</Warning>
