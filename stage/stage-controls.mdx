## Navigation sur la Scène

### Panoramique (Pan)

Déplacez la vue pour explorer la scène :

| Plateforme | Méthode |
|------------|---------|
| **Ordinateur** | Clic droit + glisser |
| **Tablette/Mobile** | Glisser avec 2 doigts |

### Zoom

Ajustez le niveau de zoom :

| Plateforme | Zoom In | Zoom Out |
|------------|---------|----------|
| **Ordinateur** | Molette haut | Molette bas |
| **Tablette/Mobile** | Pincement écartement | Pincement rapprochement |

### Réinitialiser la Vue

Boutons de réinitialisation :
- **Ajuster à la scène** : Cadre toute la scène
- **Zoom 100%** : Échelle réelle
- **Centrer** : Recentre la vue

## Grille

### Afficher/Masquer

Activez ou désactivez la grille via :
- Bouton dans la barre d'outils
- Raccourci : `G`

### Configuration de la Grille

<Tabs>
  <Tab title="Pas de Grille">
    Ajustez l'espacement :
    - **0.5m** : Très précis
    - **1m** : Standard (recommandé)
    - **1.5m** : Moyen
    - **2m** : Large
  </Tab>
  
  <Tab title="Subdivisions">
    Lignes intermédiaires pour plus de précision :
    - **Aucune** : Grille simple
    - **2 subdivisions** : Demi-pas
    - **4 subdivisions** : Quart de pas
  </Tab>
  
  <Tab title="Apparence">
    - **Couleur** : Personnalisable
    - **Opacité** : Ajustable
    - **Style** : Lignes ou points
  </Tab>
</Tabs>

## Magnétisme (Snap to Grid)

### Activer/Désactiver

- **Bouton** : Icône aimant dans la barre d'outils
- **Raccourci** : `Ctrl/Cmd + Shift + G`
- **Par défaut** : Désactivé

### Fonctionnement

Quand activé :
- Les danseurs s'accrochent aux intersections de la grille
- Facilite l'alignement précis
- Ajustable en sensibilité

<Tip>
  Le magnétisme est particulièrement utile pour créer des formations géométriques parfaites.
</Tip>

## Mode Plein Écran

### Activer le Plein Écran

Maximisez l'espace de travail :
- **Bouton** : Icône plein écran
- **Raccourci** : `F11` ou `Fn + F`
- **Double-clic** : Sur la barre de titre

### Avantages

- Plus d'espace pour la scène
- Meilleure visibilité
- Idéal pour les présentations
- Moins de distractions

<Info>
  En mode plein écran, les panneaux latéraux deviennent des tiroirs coulissants accessibles via des boutons.
</Info>

## Outils de Mesure

### Règle

Mesurez des distances sur scène :
1. Activez l'outil règle
2. Cliquez sur le point de départ
3. Cliquez sur le point d'arrivée
4. La distance s'affiche en mètres

### Rapporteur

Mesurez des angles :
1. Activez l'outil rapporteur
2. Définissez le centre
3. Définissez les deux points
4. L'angle s'affiche en degrés

## Calques et Visibilité

### Gérer les Calques

Contrôlez ce qui est affiché :

| Calque | Description | Raccourci |
|--------|-------------|-----------|
| **Danseurs** | Afficher/masquer tous les danseurs | `D` |
| **Marqueurs** | Afficher/masquer les marqueurs | `M` |
| **Rideaux** | Afficher/masquer les rideaux | `R` |
| **Grille** | Afficher/masquer la grille | `G` |

### Mode Fantôme

Affichez les positions précédentes en transparence :
- Utile pour comparer les formations
- Ajustable en opacité
- Activable par groupe de placement

## Raccourcis Clavier

| Raccourci | Action |
|-----------|--------|
| `Espace` | Pan (maintenir + glisser) |
| `+` / `-` | Zoom in/out |
| `0` | Zoom 100% |
| `F` | Ajuster à la scène |
| `G` | Toggle grille |
| `Ctrl/Cmd + Shift + G` | Toggle magnétisme |
| `F11` | Plein écran |
