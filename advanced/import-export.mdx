## Import CSV de Danseurs

### Format du Fichier

Créez un fichier CSV avec les colonnes suivantes :

```csv
prenom,nom,photo,scenes
Marie,Dupont,https://example.com/photo1.jpg,"Acte 1,Finale"
Jean,Martin,https://example.com/photo2.jpg,"Acte 1,Acte 2"
Sophie,Bernard,,Finale
```

### Colonnes

| Colonne | Requis | Description |
|---------|--------|-------------|
| `prenom` | ✅ Oui | Prénom du danseur |
| `nom` | ✅ Oui | Nom de famille |
| `photo` | ❌ Non | URL de la photo |
| `scenes` | ❌ Non | Noms des scènes (séparés par virgules) |

### Procédure d'Import

<Steps>
  <Step title="Préparer le fichier">
    Créez votre CSV avec un tableur ou éditeur de texte
  </Step>
  
  <Step title="Lancer l'import">
    Onglet Cast > **Importer CSV**
  </Step>
  
  <Step title="Sélectionner le fichier">
    Choisissez votre fichier CSV
  </Step>
  
  <Step title="Configurer les options">
    - ✅ Importer les danseurs
    - ✅ Importer les scènes (si présentes)
    - ✅ Assigner automatiquement aux scènes
  </Step>
  
  <Step title="Valider">
    Vérifiez l'aperçu et cliquez sur **Importer**
  </Step>
</Steps>

<Tip>
  Encodez votre fichier en UTF-8 pour préserver les accents et caractères spéciaux.
</Tip>

## Export de Projets

### Exporter un Projet Complet

Pour sauvegarder ou partager :

1. Menu projet > **Exporter**
2. Choisissez le format :
   - **JSON** : Données complètes
   - **ZIP** : Projet + médias
3. Téléchargez le fichier

### Contenu de l'Export

L'export inclut :
- ✅ Configuration du projet
- ✅ Tous les danseurs
- ✅ Toutes les scènes
- ✅ Tous les placements
- ✅ Groupes de placement
- ✅ Marqueurs
- ✅ Configuration des rideaux
- ❌ Fichiers audio (trop volumineux)

## Export de Scènes

### Exporter une Scène Spécifique

Pour partager une scène :

1. Sélectionnez la scène
2. Menu (⋮) > **Exporter**
3. Choisissez le format :
   - **PDF** : Vue imprimable
   - **Image (PNG)** : Capture d'écran
   - **JSON** : Données brutes

### Export PDF

Le PDF inclut :
- Vue de la scène avec placements
- Liste des danseurs
- Informations de la scène
- Légende et repères

<Info>
  L'export PDF est idéal pour distribuer aux danseurs ou à l'équipe technique.
</Info>

## Duplication de Projets

### Dupliquer un Projet

Pour créer une copie :

<Steps>
  <Step title="Accéder à la liste">
    Tableau de bord > Liste des projets
  </Step>
  
  <Step title="Menu du projet">
    Cliquez sur (⋮) > **Dupliquer**
  </Step>
  
  <Step title="Choisir les éléments">
    Sélectionnez ce qui doit être copié
  </Step>
  
  <Step title="Nommer">
    Donnez un nom à la copie
  </Step>
  
  <Step title="Créer">
    Cliquez sur **Dupliquer**
  </Step>
</Steps>

## Import de Projets

### Importer un Projet Exporté

Pour restaurer ou importer :

1. Tableau de bord > **Importer un Projet**
2. Sélectionnez le fichier JSON ou ZIP
3. Vérifiez l'aperçu
4. Cliquez sur **Importer**

<Warning>
  L'import créera un nouveau projet. Il ne fusionnera pas avec un projet existant.
</Warning>
