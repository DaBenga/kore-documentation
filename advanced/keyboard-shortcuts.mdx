## Navigation Générale

| Raccourci | Action |
|-----------|--------|
| `Ctrl/Cmd + S` | Sauvegarder |
| `Ctrl/Cmd + Z` | Annuler |
| `Ctrl/Cmd + Y` | Refaire |
| `Ctrl/Cmd + A` | Tout sélectionner |
| `Suppr` | Supprimer la sélection |
| `Échap` | Annuler l'action en cours |

## Scène (Stage)

### Navigation

| Raccourci | Action |
|-----------|--------|
| `Espace` (maintenir) | Mode panoramique |
| `+` / `-` | Zoom in/out |
| `0` | Zoom 100% |
| `F` | Ajuster à la scène |
| `F11` | Plein écran |

### Grille et Affichage

| Raccourci | Action |
|-----------|--------|
| `G` | Toggle grille |
| `Ctrl/Cmd + Shift + G` | Toggle magnétisme |
| `D` | Toggle danseurs |
| `M` | Toggle marqueurs |
| `R` | Toggle rideaux |

### Placement de Danseurs

| Raccourci | Action |
|-----------|--------|
| `←` `→` `↑` `↓` | Déplacer sélection (1px) |
| `Shift + Flèches` | Déplacer sélection (10px) |
| `Ctrl/Cmd + Clic` | Ajouter à la sélection |
| `Shift + Clic` | Sélectionner une plage |
| `Ctrl/Cmd + D` | Dupliquer la sélection |

## Timeline

### Lecture

| Raccourci | Action |
|-----------|--------|
| `Espace` | Play/Pause |
| `S` | Stop |
| `←` / `→` | Groupe précédent/suivant |
| `Shift + ←/→` | Marqueur précédent/suivant |
| `Home` | Aller au début |
| `End` | Aller à la fin |

### Édition

| Raccourci | Action |
|-----------|--------|
| `Ctrl/Cmd + N` | Nouveau groupe |
| `Ctrl/Cmd + D` | Dupliquer groupe |
| `Ctrl/Cmd + C` | Copier groupe |
| `Ctrl/Cmd + V` | Coller groupe |
| `Ctrl/Cmd + Shift + D` | Diviser groupe |
| `M` | Ajouter un marqueur |

### Navigation

| Raccourci | Action |
|-----------|--------|
| `+` / `-` | Zoom temporel in/out |
| `F` | Ajuster timeline à la vue |
| `Shift + Molette` | Défiler horizontalement |

## Projets et Scènes

| Raccourci | Action |
|-----------|--------|
| `Ctrl/Cmd + N` | Nouveau projet/scène (selon contexte) |
| `Ctrl/Cmd + O` | Ouvrir projet |
| `Ctrl/Cmd + W` | Fermer projet |
| `Ctrl/Cmd + Tab` | Scène suivante |
| `Ctrl/Cmd + Shift + Tab` | Scène précédente |

## Audio

| Raccourci | Action |
|-----------|--------|
| `Ctrl/Cmd + M` | Mute/Unmute |
| `Ctrl/Cmd + ↑/↓` | Augmenter/Diminuer volume |

## Historique

| Raccourci | Action |
|-----------|--------|
| `Ctrl/Cmd + H` | Ouvrir panneau d'historique |
| `Ctrl/Cmd + Z` | Annuler |
| `Ctrl/Cmd + Y` | Refaire |

## Aide

| Raccourci | Action |
|-----------|--------|
| `F1` | Ouvrir l'aide |
| `?` | Afficher les raccourcis |
| `Ctrl/Cmd + /` | Recherche dans l'aide |

## Conseils

<AccordionGroup>
  <Accordion title="Apprentissage" icon="graduation-cap">
    - Commencez par les raccourcis de base
    - Ajoutez-en progressivement

  <Accordion title="Productivité" icon="bolt">

    - Évitez la souris quand possible
  </Accordion>
  
  <Accordion title="Compatibilité" icon="computer">
    - `Ctrl` sur Windows/Linux
    - `Cmd` sur Mac
    - Certains raccourcis peuvent varier selon le navigateur
