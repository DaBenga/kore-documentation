## Contacter le Support

Notre équipe d'assistance est là pour vous aider.

### Par Email

<Card
  title="Envoyer un Email"
  icon="envelope"

### Informations à Fournir

Pour une assistance rapide, incluez :

- **Votre email** : Compte Koré
- **Navigateur** : Chrome, Safari, Firefox, etc.
- **Appareil** : Ordinateur, tablette, smartphone
- **Description du problème** : Soyez aussi précis que possible
- **Captures d'écran** : Si applicable
- **Étapes pour reproduire** : Comment le problème se produit

## Signaler un Bug

### Procédure

Si vous rencontrez un bug :

<Steps>
  <Step title="Vérifiez la FAQ">Le problème est peut-être déjà documenté</Step>

<Step title="Reproduisez le bug">
  Essayez de reproduire le problème de manière cohérente
</Step>

<Step title="Collectez les informations">
  - Navigateur et version - Étapes pour reproduire - Captures d'écran - Messages
  d'erreur
</Step>

  <Step title="Envoyez un rapport">
    Utilisez le **formulaire de contact** accessible en bas à droite de
    l'application pour envoyer un rapport de bug ou une demande de
    fonctionnalité.
  </Step>
</Steps>
