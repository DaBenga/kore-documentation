## Compte et Connexion

<AccordionGroup>
  <Accordion title="Comment créer un compte ?" icon="user-plus">
    1. Rendez-vous sur [kore.app](https://kore.app)
    2. Cliquez sur **S'inscrire**
    3. Renseignez votre email et mot de passe
    4. Validez via l'email de confirmation
  </Accordion>
  
  <Accordion title="J'ai oublié mon mot de passe" icon="key">
    1. Sur la page de connexion, cliquez sur **Mot de passe oublié**
    2. Entrez votre adresse email
    3. Suivez les instructions reçues par email
    4. Créez un nouveau mot de passe
  </Accordion>
  
  <Accordion title="Puis-je changer mon email ?" icon="envelope">
    Oui, dans les paramètres de votre compte :
    1. Menu utilisateur > **Paramètres**
    2. Section **Compte**
    3. Modifiez votre email
    4. Validez via le nouvel email
  </Accordion>
</AccordionGroup>

## Projets

<AccordionGroup>
  <Accordion title="Combien de projets puis-je créer ?" icon="folder">
    Il n'y a pas de limite au nombre de projets que vous pouvez créer.
  </Accordion>
  
  <Accordion title="Puis-je dupliquer un projet ?" icon="copy">
    Oui :
    1. Liste des projets > Menu (⋮)
    2. Sélectionnez **Dupliquer**
    3. Choisissez les éléments à copier
    4. Nommez la copie
  </Accordion>
  
  <Accordion title="Comment supprimer un projet ?" icon="trash">
    ⚠️ La suppression est irréversible :
    1. Menu projet (⋮) > **Supprimer**
    2. Tapez le nom du projet pour confirmer
    3. Cliquez sur **Supprimer définitivement**
  </Accordion>
  
  <Accordion title="Mes projets sont-ils sauvegardés automatiquement ?" icon="floppy-disk">
    Oui, KORE sauvegarde automatiquement toutes les 30 secondes et à chaque action majeure.
  </Accordion>
</AccordionGroup>

## Danseurs

<AccordionGroup>
  <Accordion title="Comment ajouter plusieurs danseurs rapidement ?" icon="users">
    Utilisez l'import CSV :
    1. Préparez un fichier CSV avec colonnes : `prenom`, `nom`, `photo`
    2. Onglet Cast > **Importer CSV**
    3. Sélectionnez votre fichier
    4. Validez l'import
    
    [Guide complet](/dancers/add-dancers)
  </Accordion>
  
  <Accordion title="Les photos sont-elles obligatoires ?" icon="image">
    Non, les photos sont optionnelles. Sans photo, les initiales du danseur sont affichées.
  </Accordion>
  
  <Accordion title="Puis-je modifier un danseur après création ?" icon="pen">
    Oui, cliquez sur le danseur puis sur l'icône d'édition (crayon) pour modifier toutes ses informations.
  </Accordion>
</AccordionGroup>

## Scènes et Timeline

<AccordionGroup>
  <Accordion title="Qu'est-ce qu'un groupe de placement ?" icon="layer-group">
    Un groupe de placement capture les positions de tous les danseurs à un moment donné. Il permet de créer différentes formations dans le temps.
  </Accordion>
  
  <Accordion title="Puis-je ajouter de la musique ?" icon="music">
    Oui, pour chaque scène :
    1. Sélectionnez la scène
    2. Cliquez sur **+ Ajouter Audio**
    3. Formats supportés : MP3, WAV, M4A
    
    [Guide audio](/scenes/audio)
  </Accordion>
  
  <Accordion title="Comment synchroniser avec la musique ?" icon="clock">
    1. Ajoutez un fichier audio à la scène
    2. La waveform s'affiche sur la timeline
    3. Positionnez le playhead au moment souhaité
    4. Créez vos groupes de placement
    5. Ils seront synchronisés avec la musique
  </Accordion>
</AccordionGroup>

## Collaboration

<AccordionGroup>
  <Accordion title="Puis-je partager un projet ?" icon="share">
    Oui :
    1. Bouton **Partager** dans le projet
    2. Entrez l'email du collaborateur
    3. Choisissez le rôle (Lecteur, Éditeur, Administrateur)
    4. Envoyez l'invitation
    
    [Guide permissions](/projects/permissions)
  </Accordion>
  
  <Accordion title="Combien de collaborateurs puis-je inviter ?" icon="users">
    Il n'y a pas de limite au nombre de collaborateurs par projet.
  </Accordion>
  
  <Accordion title="Peut-on travailler en même temps ?" icon="users-gear">
    Oui, KORE supporte la collaboration en temps réel. Les modifications des autres utilisateurs sont visibles instantanément.
  </Accordion>
</AccordionGroup>

## Mobile

<AccordionGroup>
  <Accordion title="KORE fonctionne-t-il sur mobile ?" icon="mobile">
    Oui, KORE est optimisé pour mobile et tablette :
    - iPhone/iPad (iOS 15+)
    - Android (10+)
    - Interface adaptative
    - Gestes tactiles
    
    [Guide mobile](/mobile/overview)
  </Accordion>
  
  <Accordion title="Puis-je créer un projet sur mobile ?" icon="tablet">
    Oui, toutes les fonctionnalités sont disponibles sur mobile. Cependant, l'expérience est optimale sur tablette pour le travail de création.
  </Accordion>
  
  <Accordion title="Les gestes tactiles sont-ils différents ?" icon="hand">
    Oui, KORE utilise des gestes spécifiques :
    - Pincement pour zoomer
    - 2 doigts pour panoramique
    - Appui long pour menu contextuel
    
    [Guide gestes](/mobile/gestures)
  </Accordion>
</AccordionGroup>

## Technique

<AccordionGroup>
  <Accordion title="Quels navigateurs sont supportés ?" icon="browser">
    - **Chrome** : Version 90+ (recommandé)
    - **Safari** : Version 14+
    - **Firefox** : Version 88+
    - **Edge** : Version 90+
  </Accordion>
  
  <Accordion title="Mes données sont-elles sécurisées ?" icon="lock">
    Oui :
    - Connexion HTTPS sécurisée
    - Données chiffrées
    - Sauvegardes automatiques
    - Conformité RGPD
  </Accordion>
  
  <Accordion title="Puis-je travailler hors ligne ?" icon="wifi-slash">
    Non, KORE nécessite une connexion internet pour fonctionner. Les modifications sont sauvegardées en temps réel sur le cloud.
  </Accordion>
  
  <Accordion title="Quelle taille maximale pour les fichiers audio ?" icon="file-audio">
    - **Recommandé** : Moins de 15 MB
    - **Maximum** : 50 MB
    - **Format** : MP3 (192-320 kbps) recommandé
  </Accordion>
</AccordionGroup>

## Tarifs et Abonnement

<AccordionGroup>
  <Accordion title="KORE est-il gratuit ?" icon="dollar">
    KORE propose un plan gratuit avec fonctionnalités de base. Des plans premium sont disponibles pour des fonctionnalités avancées.
  </Accordion>
  
  <Accordion title="Puis-je annuler mon abonnement ?" icon="xmark">
    Oui, vous pouvez annuler à tout moment depuis les paramètres de votre compte. Vos projets restent accessibles en lecture seule.
  </Accordion>
</AccordionGroup>
